```python
    # –ü–µ—Ä–µ–∫–ª—é—á–µ–Ω–∏–µ –∞–Ω–∏–º–∞—Ü–∏–π –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç –≤–≤–æ–¥–∞

    else:
        player.play_animation("idle")
    
    player.update()

    game.screen.blit(player.image, player.rect)

game.run(update, draw)
```
